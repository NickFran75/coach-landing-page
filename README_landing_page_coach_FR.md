# Landing Page Coach --- Site Web Professionnel 

## Démo en ligne
https://coach-landing-page1.vercel.app/

## Présentation du projet

Ce projet est une **landing page moderne destinée aux coachs,
consultants et indépendants** souhaitant disposer d'une présence
professionnelle en ligne.

Le site démontre comment un professionnel peut :

-   présenter clairement ses services
-   capter des prospects
-   permettre la prise de rendez‑vous en ligne
-   renforcer la confiance grâce aux témoignages clients
-   partager son expertise via une section conseils ou blog

Ce projet a été réalisé dans le cadre d'un **portfolio de développement
web** afin de démontrer la création d'un site professionnel prêt à être
utilisé par un business réel.

# Fonctionnalités principales

## Présentation des services

Une section dédiée permet de présenter les différents services proposés
par le coach avec :

-   titre du service
-   description
-   bénéfices pour le client
-   bouton d'appel à l'action

------------------------------------------------------------------------

## Témoignages clients

Une section témoignages permet d'ajouter des avis clients afin de
renforcer la crédibilité du site.

Les témoignages sont affichés sous forme de **cartes visuelles
modernes** avec :

-   nom
-   rôle / activité
-   avis
-   éventuellement une note

------------------------------------------------------------------------

## Formulaire de contact

Le site contient un **formulaire de contact fonctionnel** permettant aux
visiteurs d'envoyer une demande.

Champs typiques :

-   nom
-   email
-   téléphone (optionnel)
-   message

------------------------------------------------------------------------

## Prise de rendez‑vous

Le site intègre un système de **prise de rendez‑vous avec Calendly**.

Cela permet aux visiteurs de réserver directement :

-   un appel découverte
-   une session de coaching
-   un rendez‑vous de consultation

------------------------------------------------------------------------

## Section conseils / blog

Une section permet de publier des **articles ou conseils** afin de :

-   démontrer l'expertise
-   améliorer le référencement
-   apporter de la valeur aux visiteurs

# Installation du projet

Cloner le repository :

    git clone https://github.com/votre-utilisateur/coach-landing-page.git

Se placer dans le dossier du projet :

    cd coach-landing-page

Installer les dépendances :

    npm install

Lancer le serveur de développement :

    npm run dev

------------------------------------------------------------------------

# Déploiement

Le site peut être déployé facilement avec **Vercel**.

Étapes :

1.  pousser le projet sur GitHub
2.  connecter le repository à Vercel
3.  lancer le déploiement automatique

Une URL publique sera générée.
